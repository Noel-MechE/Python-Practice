# Single Word Anagram Finder

I built this program as part of my learning journey toward 
the Snap Engineering Academy 2026 application. It helped me 
understand core Python concepts such as sorting, string methods, 
and for loops, which are essential for repetitive tasks and 
optimization.

## What the Program Does

The program is loaded with an English dictionary containing 
thousands of words. It accepts a word or name from the user 
and searches through the entire dictionary to find all words 
that contain the same letters as the input. Matching words 
are collected into a list and displayed to the user.

Two words are anagrams if they contain the exact same letters 
in any order. For example Noel and Leon are anagrams, and so 
are Noel and Elon!

## How to Run

Make sure these three files are in the same folder:
- noel_anagrams.py
- load_dictionary.py
- 2of4brif.txt

Then run:
python noel_anagrams.py
